Este site foi criado com o intuito de simular o catálogo de uma loja de games, onde podemos realizar todo o CRUD (Create, Read, Update, Delete) dos games que nós quisermos.

Temos ao todo 6 telas, cada uma com uma função do CRUD e utilizamos a biblioteca SQLite para implementar o nosso banco de dados em SQL. Em relação às linguagens de programação utilizadas, usamos o Javascript para fazermos algumas interações no nosso front-end e o Python para o back-end do site.
